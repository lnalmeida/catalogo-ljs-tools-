## Moment.js - ajuda a gerenciar hora e data de forma eficaz ao trabalhar com diferentes fusos horários, chamadas de API, idiomas locais, etc.
link: https://momentjs.com

## Masonry - É uma biblioteca de layout de grade JavaScript. Funciona colocando os elementos na posição ideal com base no espaço vertical disponível, como um pedreiro colocando pedras em uma parede. Você provavelmente já o viu em uso em toda a Internet.
link: https://masonry.desandro.com

## Parsley - É uma biblioteca de validação de formulário javascript. Ele ajuda você a fornecer feedback aos usuários sobre o envio do formulário antes de enviá-lo ao servidor. Ele economiza largura de banda, carga do servidor e tempo para seus usuários. NOTA: A validação do formulário Javascript não é necessária e, se usada, não substitui a validação forte do servidor de back-end.
link: http://parsleyjs.org

## Omniscient - s uma biblioteca JS que fornece abstração do componente React para renderização imediata de cima para baixo que abrange dados imutáveis. Esta biblioteca pode ajudá-lo a construir seu projeto perfeitamente, pois é otimizado e oferece recursos interessantes.
link: http://omniscientjs.github.io

## Popper.js - foi criado para tornar mais fácil posicionar popovers, menus suspensos, dicas de ferramentas e outros elementos contextuais que aparecem perto de um botão ou outros elementos semelhantes.
link: https://popper.js.org

## Three.js - e tornar seu design 3-D agradável. Ele usa WebGL para renderizar cenas em navegadores modernos. Use outros renderizadores CSS3, CSS2 e SVH se usar o IE 10 e versões anteriores.
link: https://threejs.org

## screenfull - Wrapper simples para uso em vários navegadores da API JavaScript Fullscreen, que permite trazer a página ou qualquer elemento para tela inteira.
link: https://github.com/sindresorhus/screenfull

## Lit - Lit é uma biblioteca simples para a construção de componentes da web rápidos e leves.
link: https://lit.dev

## Voca.js - A biblioteca Voca oferece funções úteis para tornar as manipulações de strings confortáveis: change case, trim, pad, slugify, latinise, sprintf'y, truncate, escape e muito mais. O design modular permite carregar toda a biblioteca ou funções individuais para minimizar a criação de aplicativos. A biblioteca foi totalmente testada, bem documentada e com suporte de longo prazo.
link: https://vocajs.com
